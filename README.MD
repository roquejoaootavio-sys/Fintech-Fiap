# Fintech - Controle de Movimentações Financeiras

## Sobre o Projeto

Este projeto foi desenvolvido como atividade acadêmica da disciplina de Front-End da FIAP.

O objetivo foi criar uma interface web para uma Fintech, simulando uma tela de cadastro e gerenciamento de movimentações financeiras.

## Funcionalidades da Tela

* Visualização de indicadores financeiros
* Cadastro de movimentações
* Seleção de categoria e forma de pagamento
* Campo para observações
* Layout moderno e intuitivo
* Interface responsiva para dispositivos móveis

## Tecnologias Utilizadas

* HTML5
* CSS3
* Tailwind CSS
* Font Awesome

## Responsividade

O projeto foi desenvolvido utilizando classes responsivas do Tailwind CSS, permitindo adaptação para computadores, tablets e smartphones.
